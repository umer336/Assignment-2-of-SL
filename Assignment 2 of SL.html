Question#1 Difference between Var, Let and Const???

Ans: Var

The var statement declares a variable. Variables are containers for storing information. Creating a variable in JavaScript is called "declaring" a variable. Examples are: var id =10; , var name = "Umar Abdul Rehman"; etc. These variable are declares in a tag called 'script'. Examples of var are performed in Console.


Examples Explanation

//Example 1 In this example we declares three variables with same name 'x' then its value is changed when we declares it multiple times. These examples are define the different scopes of variable.
var x = 2;
// Here x is 2
{
var x = 10 ;
// Here x is 10
}
// Here x is 0
var x = 0;
console.log("Value of x is "+x);

//Example 2 In this example we declares two variables with same name 'id' then we its value does not changed when we declares same variable in function.
var id = 3;
// code here can use id
function myFunction() {
// code here can also use id
var id = 5;
}
console.log("Value of id is "+id);

Let

Let allows you to declare variables that are limited to a scope of a block statement, or expression on which it is used, unlike the var keyword, which defines a variable globally, or locally to an entire function regardless of block scope


Examples Explanation

Example 1 In this example we define two functions of 'VarTest' and 'VarLet' then its value is changed in varTest when we declares it multiple times but in varLet function its value does not changed.
function varTest() {
var x = 1;
{
var x = 2; // same variable!
console.log(x); // 2
} console.log(x); // 2
} function letTest() {
let x = 1;
{
let x = 2; // different variable
console.log(x); // 2
}
console.log(x); // 1
}
Example 2
let x = 1;
if (x === 1) {
let x = 2;
console.log(x);
// expected output: 2
}
console.log(x);
// expected output: 1

Const

The const declaration creates a read-only reference to a value. It does not mean the value it holds is immutable, just that the variable identifier cannot be reassigned. A constant cannot share its name with a function or a variable in the same scope.
